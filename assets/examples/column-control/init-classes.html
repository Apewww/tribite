Responsive example Assigned class control

This example exactly matches the functionality of the class control example but in this case the classes are assigned using the columns.className option.

The Javascript shown below is used to initialise the table shown in this example:

$('#example').DataTable({ ajax: '../../../../examples/ajax/data/objects.txt', columns: [ { data: 'name', className: 'all' }, { data: 'position', className: 'min-phone-l' }, { data: 'office', className: 'min-tablet' }, { data: 'start_date', className: 'never' }, { data: 'salary', className: 'desktop' }, { data: 'extn', className: 'none' } ], responsive: true });
new DataTable('#example', { ajax: '../../../../examples/ajax/data/objects.txt', columns: [ { data: 'name', className: 'all' }, { data: 'position', className: 'min-phone-l' }, { data: 'office', className: 'min-tablet' }, { data: 'start_date', className: 'never' }, { data: 'salary', className: 'desktop' }, { data: 'extn', className: 'none' } ], responsive: true });
