Nama Aplikasi: Tribite
Deskripsi: Merupakan Point of Sale, untuk memesan makanan, dapat memesan makanan, dan mengatur data untuk akses staff


Tujuan:
Migrasikan project ini menjadi next.js (App Router) dan database menggunakan supabase, variable ada pada .env.local, dan untuk framework css menggunakan tailwindcss, untuk styling menggunakan shadcn/ui
